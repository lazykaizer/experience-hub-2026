## 🌟 Overview

**GfG ExperienceHub 2026** is a dedicated one-month initiative powered by **GeeksforGeeks** (GfG Connect). It is designed for students, freshers, and professionals to openly share their authentic tech journeys. 

This isn't just another tech blog; it's a mission control for real-world career narratives. From cracking high-stakes interviews to navigating complex campus lives, ExperienceHub is where raw experiences turn into community inspiration.

---

## 🔥 Key Pillars of the Platform

### 1. High-Impact Hero & Experience Marquee ⚡
* **Live Event Tracking:** Interactive countdown and event date badges.
* **Real-time Stats:** Full-width scrolling marquee capturing the pulse of the GfG community involvement.

### 2. The 5 Content Power-Themes 📚
Every submission is categorized into specialized, high-visibility hashtags:
* **#campusexperiences:** Real college life & projects.
* **#interviewexperiences:** Round-by-round career breakthroughs.
* **#workexperiences:** Deep dives into professional growth.
* **#admissionexperiences:** Honest counseling and admission stories.
* **#examexperiences:** Strategy, resources, and exam-day narratives.

### 3. Rewards & Gamification 🏆
* **Daily Cash Rewards:** Active participation tracking for ₹500 daily prizes.
* **Social Media Integration:** Swag eligibility through verified LinkedIn/X sharing.

### 4. Quality Governance (The Rules Engine) ⚖️
* **Strict Authenticity:** Built-in guidelines to prevent AI-generated or plagiarized content.
* **Visual Focus:** Guidance for including campus photos and screenshots to boost reach.

## 🎨 Design Philosophy: The "Neo-GfG" Aesthetic

ExoerienceHub features a **Neobrutalist** design system that is bold, confident, and developer-first:
* **High Contrast:** Rich black borders (`neo-border`) on muted cream backgrounds.
* **Textured Surfaces:** Subtle grain overlays and dot-grid backgrounds (`dot-pattern`) for a premium tactile feel.
* **Confidential Widget:** A sleek, "Notion-inspired" floating card for creator connectivity.
* **Premium Micro-interactions:** Bouncy entrance animations and high-fidelity hover glows.

## 🚦 Getting Started

### 1. Clone & Install
```bash
git clone https://github.com/lazykaizer/experience-hub-2026.git
cd experience-hub-2026
npm install
```

### 2. Run Locally
```bash
npm run dev
```
Explore the local build at [http://localhost:5173](http://localhost:5173).

### 3. Build for Production
```bash
npm run build
```

## 🛡️ Security & Performance

* **Modular Codebase:** Each section is a self-contained React component for rapid scaling.
